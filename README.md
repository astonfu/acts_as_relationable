# ActsAsRelationable
Make Active Recored acts as relationable.
## Usage
How to use my plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'acts_as_relationable'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install acts_as_relationable
```
